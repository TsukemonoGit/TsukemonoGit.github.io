# 要件書 — tsukemonogit.github.io

## 目的

mono（₍ ･ᴗ･ ₎）の活動記録を公開するポートフォリオ的なサイト。
商用目的ではない。個人的な活動ログをまとめたサイトである。

## 構成

- **データ元**: `static/data.csv`（プロジェクト情報・アカウント情報）
- **ビルド**: SvelteKit + Tailwind CSS
- **ホスティング**: GitHub Pages（`static` アダプター）

## 機能要件

### 1. データ処理

- `static/data.csv` を読み込み、`src/lib/generated/data.json` に変換

## 技術制約

- Svelte 5（runes mode）
- Tailwind CSS v4
- TypeScript
- GitHub Pages 対応（static アダプター）
- `static/data.csv` は変更不可
- カラーはインラインで直接指定しない。`app.css` に Tailwind テーマ変数として定義し、クラス経由で使用すること。

## ビルドコマンド

```bash
npm i
npm run build
```
